# Governating Linodes

Linode has a feature called StackScripts,
which can be used to run small fragments of code on first boot.

Linode also has an API and a Python CLI tool,
which can be used to provision new VPS servers which run a specific StackScript on boot.

Combining these two things allows fully automated provisioning of new Governated servers.


## The Governator StackScript

Your StackScript contents depend on the public URL of your Governator Server,
so the correct script for your needs is autogenerated using same systems as everything else.

After `g8r init` and `g8r make`
you should have a script in `tree/linode/StackScript.sh` which you can add to your Linode account.

You will need to make a note of the ID Linode assigns to the StackScript and configure
`g8r_stackscript_id=XXX` in `tree/vars/g8r.vars` to match.
