Usage: g8r add-host <host_name> <host_type> [<domain.tld>]

Add a new host to your Governator for management.

The `host_type` must match one of the directories in `tree/skeletons/`.

Set the environment variables IPv4 and/or IPv6 first if you already know the
public IPs of this host.

Examples:

    g8r add-domain www web
    g8r add-domain www web example.org

    IPv4=10.1.2.3 IPv6=::1 g8r add-domain www web
