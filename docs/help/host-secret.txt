Usage: g8r host-secret <host>
       g8r host-dir <host>

Print just the host's secret or configuration directory path to stdout,
with no other output. Mostly used internally by `g8r`, but may also be
useful in custom scripts and pipelines.

<host> may be a short hostname, a FQDN, or the secret itself. If you
pass a secret that already matches an existing host directory, it is
echoed back unchanged.

If the host cannot be found, exits non-zero with an error on stderr.

Examples:

    g8r host-secret www
    SECRET=$(g8r host-secret www.example.org)
    g8r backups "$(g8r host-secret www)"

    ls -l $(g8r host-dir www.example.org)
